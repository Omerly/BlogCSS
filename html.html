¿Qué es HTML?

A lo largo de nuestra vida todos hemos tenido interacción con diferentes sitios, páginas o aplicaciones web, ya sea para buscar información referente a un tema, para interactuar con nuestros amigos, entre muchas otras cosas más. Incluso, en este mismo momento lo estás haciendo, mientras lees nuestro artículo. Si has llegado hasta aquí, es porque te interesa saber la definición de HTML y para ello será necesario comprender cómo está relacionado el término HTML con lo que ves en tu navegador web (Ej. Google Chrome, Mozilla Firefox, Safari, etc).

HTML no es un lenguaje de programación, lo primero que debes tener muy en claro, HTML es un lenguaje de marcado de hipertexto o “HyperText Markup Language” por el desarrollo de sus iniciales en inglés, básicamente este lenguaje se escribe en su totalidad con elementos, estos elementos están constituidos por etiquetas, contenido y atributos. HTML es un lenguaje que interpreta el navegador web para mostrar los sitios o aplicaciones web tal y como estamos acostumbrados.
